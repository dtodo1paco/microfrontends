## Prerequisites

- npm >=7.20.0
- node >=16.6.2

## Install

```sh

```

## Usage

### development
It will start development server on port 3001
```sh
yarn start
```
### production
It will start production server on port 3001
```sh
yarn build 
yarn serve
```
